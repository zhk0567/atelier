# 图 4–10 AI 生图提示词

**用法**：整段复制到即梦 / 可灵 / 通义万相 / Midjourney 等。若生成文字乱码，用图生图或 PS 改字；谱例类（图 9）更推荐 MuseScore 导出后 AI 只加标注框。

**统一后缀**（可追加到每条提示词末尾）：

```
教育类信息图，扁平插画，白底，中文标注清晰，无水印，无真人脸，16:9横图，高清，排版整齐，音乐教学风格，浅色点缀蓝绿色。
```

## 图 4 · fig-04-scale-and-keys.png

```
制作一张音乐教学信息图，横版。上半部分：C大调简谱音阶，横向排列数字 1 2 3 4 5 6 7，后面是高音点的 1（表示高八度do），每个数字下方标注唱名 do re mi fa sol la si do，再下方标注音名 C D E F G A B C。下半部分：钢琴键盘俯视图（一个八度加下一个C），用彩色高亮标出7个白键与简谱1到7的对应箭头。标题：「1=C 时，数字与琴键」。风格简洁、适合博客。
```

## 图 5 · fig-05-octave-dots.png

```
制作一张简谱教学示意图，横版。左侧三行对比：同一个音「5」分别写作中音5、高音（数字左上有一个点）、低音（数字左下有一个点），每行旁用中文标注「中音区」「高八度」「低八度」。右侧：竖向八度阶梯图，三层，每层写 1 2 3 4 5 6 7，层与层之间用箭头连接，说明「上加一点 = 跳到上一组1-7」。标题：「高低音点」。白底扁平插画，中文清晰。
```

## 图 6 · fig-06-rhythm.png

```
制作一张简谱节奏记号教学图，横版，分三栏。第一栏标题「音的长短」：展示四分音符（单独数字1）、八分音符（数字2下面一条下划线）、十六分音符（数字3下面两条下划线），分别标注「1拍」「半拍」「1/4拍」。第二栏标题「附点」：数字1右边有附点，标注「延长原时值一半」。第三栏标题「休止」：数字0表示休止符，标注「这一拍不唱但要数拍」。白底，黑色简谱数字，教学信息图风格。
```

## 图 7 · fig-07-time-measures.png

```
制作一张简谱拍号与小节教学图，横版。左上：大字拍号「3/4」，旁注「每小节3拍，以四分音符为一拍」。下方：三个并排的小节框，每个小节内用简谱数字示例（如 1 2 3 | 5 6 1 | 2 3 5），小节之间用竖线分隔，标注「小节线」。在第一个小节下方标注强弱「强 弱 弱」。标题：「拍号与小节」。教育信息图，白底，中文标注。
```

## 图 8 · fig-08-key-accidentals.png

```
制作一张简谱调号与变化音教学图，横版，上下两部分。上半部分标题「调号」：左边谱头写 1=C，箭头指向 do=C；右边谱头写 1=G，箭头指向 do=G，中间文字「数字仍是1 2 3…，换调后实际音高整体移动」。下半部分标题「临时升降号」：三个例子并排，#4（升fa）、b7（降si）、还原号加5，每个用箭头指向数字并简短中文说明。白底扁平风格，适合音乐入门博客。
```

## 图 9 · fig-09-twinkle-annotated.png

```
制作一张《小星星》简谱教学图，横版，1=C，4/4拍。显示两行简谱旋律（一闪一闪亮晶晶相关），数字简谱，带中文歌词在数字下方。用彩色圆圈和编号标注：①拍号4/4 ②调号1=C ③一处高音点 ④一处八分音符下划线 ⑤小节线。风格像教材插图，白底，谱面整齐。注意数字和中文标注准确、可读。
```
