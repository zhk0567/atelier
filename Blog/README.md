# 博客文章

个人博客 Markdown 源稿。每篇文章一个子目录：`index.md` + `images/`。

| 文章 | 状态 | 说明 |
|------|------|------|
| [认识简谱](./认识简谱/index.md) | 可发布 | 10 张配图在 `认识简谱/images/` |
| [配图清单](./认识简谱/配图清单.md) | — | 文件名对照 |
| [AI 生图提示词（归档）](./认识简谱/_meta/AI生图提示词.md) | 归档 | 图 4–10 生成用，已完成 |

**目录约定**

```
Blog/
└── <文章名>/
    ├── index.md      # 正文
    ├── images/       # 文内配图
    ├── 配图清单.md   # 可选
    └── _meta/        # 生成过程、提示词等（非正文）
```

发布到站点时，可将 `images/` 镜像到 `static/blog/<slug>/`（见根目录 README）。
