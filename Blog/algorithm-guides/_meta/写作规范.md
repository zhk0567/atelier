# Algorithm 系列 · 指南体例规范

## 定位

将 [zhk0567/Algorithm](https://github.com/zhk0567/Algorithm) 中**专题级** `notes.md` 整理为站点可读的双语教程：一篇博文内同时覆盖 **Python** 与 **C++** 镜像路径，**不包含** `problems/leetcode/<题号>_*/` 单题博文。

Study 仓库为源码与笔记来源；正文须**扩写**，不得整段复制索引表凑字数。

## 单页结构（顺序固定）

1. **导读** — 专题定位、面试/刷题关系  
2. **预备知识** — 前置结构与复杂度（`>` 引用块标版本/环境）  
3. **Study 仓库对照** — `topic_path`、PowerShell 运行 `solution.py` / `g++`（`-LiteralPath`）  
4. **基础篇** — `###` 标题与 `guide-toc` 的 `essentials` 一致  
5. **Python 实现** — 摘录并讲解 `python/...` 代码  
6. **C++ 实现** — 对照 `cpp/...`；必要时提及 `cpp/include/alg_std.hpp`  
7. **练习与延伸** — 链到 Study 题解目录或题单，不在 atelier 新建单题页  
8. **学习路径**  
9. **延伸阅读** — GitHub 与仓库内 `notes.md`  

允许的顶层 `##` 仅限以上九块。**禁止**附录、常见问题大全、实验手册式独立大块。
